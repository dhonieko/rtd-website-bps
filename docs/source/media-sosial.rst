Media Sosial
============
.. figure:: /_static/images/media-sosial.png
    :width: 80%
    :align: center
    :alt: Media Sosial
    
Pada Website BPS disajikan informasi mengenai media sosial resmi yang dimiliki atau dikelola. Harapannya dengan disajikannya informasi media sosial ini melalui Website BPS dapat disajikan sebagai bahan rujukan bagi pengguna untuk memastikan keabsahan dari media sosial yang mengatasnamakan BPS. Pada umumnya BPS mengelola beberapa media sosial diantaranya Youtube, Instagram, Twitter (X), dan Facebook.
